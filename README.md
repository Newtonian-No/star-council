# 星海理事会 (Star Council)

AI 驱动的学术论文协作写作系统。5 个专业理事协同工作，辅助计算机/AI 论文创作。

## 理事阵容

- 架构理事 — 方法设计、技术创新性
- 实验理事 — 实验方案、统计严谨性
- 文献理事 — 领域地图、相关工作
- 文字理事 — 叙事逻辑、venue 适配
- 魔鬼代言人 — 全程挑刺、模拟 reviewer

## 快速开始

```bash
bash bin/setup.sh
python bin/council.py meet "project-name" --stage ideation --topic "研究方向讨论"
```
